MindSense terms summary (as-built v1.0.0)

MindSense AI provides wellness-oriented behavioral guidance. It is not marketed as emergency intervention or clinical diagnosis.

1. Usage boundary

  • Use MindSense as wellness support, not as a replacement for licensed medical or mental-health care.
  • For urgent or immediate safety concerns, contact local emergency services.
  • In the US, call or text 988 for crisis support.

2. Scope clarity

  • Primary production navigation is Today, Regulate, and Data (Settings via profile menu).
  • Community, QA Tools, and KPI Scorecard exist in code but are not part of primary production tab navigation.
  • This website describes implemented behavior and avoids roadmap features presented as shipped.

3. Integration status notes

  • No full cloud sync pipeline is implemented in this build.
  • No full real HealthKit ingestion pipeline is implemented in this build.
  • Post-activation paywall UX is present, but live StoreKit billing is not fully wired in v1.0.0.

4. Contact

For terms and commercial questions, contact partnerships@mindsense.ai.
