        Trust, Safety, and Quality

        Explicit boundaries, local-first posture, and deterministic QA workflows.

        Wellness boundaries

        • MindSense is framed as wellness support, not emergency or diagnostic care.
        • Settings includes explicit crisis support path (`tel://988` in the US).
        • In high-load contexts, escalation guidance appears in onboarding and core surfaces.

        Data and privacy posture

        • v1.0.0 runtime is local-first (`UserDefaults` + Codable persistence).
        • Sign in with Apple is the only implemented auth path.
        • No in-app cloud sync pipeline is implemented in this build.
        • Health diagnostics are visible in-app; real HealthKit ingestion is not yet shipped.

        Quality gates

        • `Scripts/design_qa.sh` orchestrates contrast, snapshots, dynamic type, and latency checks.
        • UI tests enforce key CTA visibility and performance budgets.
        • Marketing screenshots are regenerated from a dedicated UI test export.
        • Copy on this site is constrained to implemented behavior, not speculative roadmap claims.

        Implemented but not wired to primary production tab navigation: Community, QA Tools, and KPI Scorecard.

        FAQ

        Common diligence questions

        Is MindSense a medical device?

        No. MindSense is currently positioned as a wellness product and does not replace licensed clinical or emergency care.

        What is implemented in the current production navigation?

        Today, Regulate, and Data are wired as primary runtime tabs, with Settings reachable from the profile menu.

        How are recommendations framed to avoid overconfidence?

        The app surfaces recommendation confidence, data coverage, signal diagnostics, and low-coverage behavior prompts.

        Does v1.0.0 include live billing, cloud sync, or full HealthKit ingestion?

        No. The post-activation paywall is presentation-level, cloud sync is not implemented, and health diagnostics are currently simulated.

        What should users do in urgent crisis situations?

        Use local emergency services immediately. In the United States, call or text 988 for crisis support.
