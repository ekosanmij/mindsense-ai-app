# MindSense Website

Single-page marketing website for MindSense AI.

## Quick Start

From project root:

```bash
python3 -m http.server 8080 --directory Website
```

Then open:

`http://127.0.0.1:8080/index.html`

## Update Download Links

Edit:

- `Website/script.js`

Update these values:

- `downloadUrl`
- `testFlightUrl`
- `privacyUrl`
- `supportEmail`
