## Geometry
- Construction grid: 24x24 equivalent.
- Safe clearspace: 25% of icon width on all sides.
- Pulse stroke weight: ~8.3% of icon width.
- Dot diameter: pulse stroke x 0.96.

## App icon notes
- Background gradient:

  - Bottom: `#070B10`
- Mark scale in icon: ~66% of canvas.
- Optical center: nudged slightly upward.
- Tinted variant: monochrome symbol with transparent background.
